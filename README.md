# Traitly

Traitly is an open-source computer vision toolkit for high-throughput fruit and locule phenotyping.  Analyze traits such as **fruit morphology** (size, shape), **internal anatomy** (locule number, shape, size, symmetry), **pericarp thickness**, and extract **tissue color** metrics (RGB, LAB, HSV, grayscale).
